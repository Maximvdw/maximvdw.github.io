WisaReg

My final work at my highschool was a project called WisaReg. WisaReg is a database migration tool to convert data from the Wisa database to other internal databases. The tool aimed to help the ICT administrators to automatically create new user accounts in LDAP, Chamilio, Smartschool and other internal databases.

The project was awarded the best final work of 2013 by Don Bosco Hoboken.
